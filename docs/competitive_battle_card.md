## Discovery Questions

### Qualifying Questions

1. **Hallucination Cost:**
   "What happens when your AI gives wrong information? What's the cost?"
   - If answer = "Not much" ‚Üí Wrong fit
   - If answer = "Could be catastrophic" ‚Üí Great fit

2. **Correctness vs Creativity:**
   "Do you need creativity or correctness from AI?"
   - If "Creativity" ‚Üí Consider LLMs
   - If "Correctness" ‚Üí Perfect for AII

3. **Performance Requirements:**
   "How fast does your AI need to respond?"
   - If "A few seconds is fine" ‚Üí Any AI works
   - If "Real-time (milliseconds)" ‚Üí AII advantage

4. **Explainability:**
   "Do you need to explain AI decisions to regulators/customers?"
   - If "No" ‚Üí Less differentiation
   - If "Yes" ‚Üí Major AII advantage

5. **Budget:**
   "What's your current AI spend (API + infra)?"
